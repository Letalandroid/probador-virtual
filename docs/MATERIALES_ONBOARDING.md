## Bienvenida

¡Bienvenido al **Probador Virtual con IA**! Este documento te guiará en tus primeros pasos con la plataforma, ya seas un usuario cliente, administrador o desarrollador.

### ¿Qué es el Probador Virtual?

Es una aplicación web que permite a los clientes visualizar cómo se verían las prendas de ropa en ellos antes de realizar una compra, utilizando inteligencia artificial para crear simulaciones realistas.

### Objetivos de este Onboarding

- Familiarizarte con la plataforma
- Aprender las funcionalidades principales
- Comprender cómo usar el probador virtual
- Conocer las herramientas de administración (si aplica)
- Acceder a recursos de ayuda

## Onboarding para Usuarios Clientes

### Paso 1: Crear tu Cuenta

1. **Accede a la aplicación**
   - Abre tu navegador
   - Visita la URL de la aplicación

2. **Registro**
   - Haz clic en **"Iniciar Sesión"** o **"Registrarse"**
   - Completa el formulario:
     - Email
     - Contraseña (mínimo 8 caracteres)
     - Nombre completo
   - Haz clic en **"Registrarse"**

3. **Verificación**
   - Recibirás un mensaje de confirmación
   - Ya puedes iniciar sesión

### Paso 2: Explorar el Catálogo

1. **Navegar por categorías**
   - En la página de inicio, verás categorías destacadas
   - Haz clic en cualquier categoría para ver sus productos

2. **Buscar productos**
   - Usa la barra de búsqueda en el header
   - Filtra por género (Mujeres, Hombres)
   - Filtra por categoría

3. **Ver detalles**
   - Haz clic en cualquier producto
   - Revisa la información completa

### Paso 3: Usar el Probador Virtual

1. **Seleccionar una prenda de torso**
   - Solo funcionan: Vestidos, Blusas, Sudaderas, Chaquetas, Camisetas
   - Haz clic en **"Probar con IA"**

2. **Preparar tu foto**
   - Lee las instrucciones en la pantalla
   - Asegúrate de tener:
     - Buena iluminación
     - Encuadre de mitad del cuerpo hacia arriba
     - Posición frontal

3. **Subir y procesar**
   - Sube tu foto
   - Haz clic en **"Probar Prenda"**
   - Espera el resultado (5-15 segundos)

4. **Ver resultado**
   - Revisa cómo se ve la prenda en ti
   - Prueba otras prendas con la misma foto

### Paso 4: Gestionar tu Perfil

1. **Acceder al perfil**
   - Haz clic en el icono de usuario
   - Selecciona **"Mi Perfil"**

2. **Completar información**
   - Agrega tu teléfono
   - Completa tu dirección
   - Guarda los cambios

### Checklist de Primeros Pasos (Cliente)

- [ ] Crear cuenta
- [ ] Explorar al menos 3 categorías
- [ ] Buscar un producto específico
- [ ] Ver detalles de un producto
- [ ] Probar el probador virtual con una prenda
- [ ] Completar el perfil
- [ ] Cambiar el tema (claro/oscuro)

## Onboarding para Administradores

### Paso 1: Acceso al Panel de Administración

1. **Iniciar sesión como administrador**
   - Usa tus credenciales de administrador
   - Si no tienes acceso, contacta al administrador principal

2. **Acceder al panel**
   - Haz clic en tu perfil
   - Selecciona **"Panel de Administración"**
   - O accede directamente a `/admin`

### Paso 2: Familiarizarte con el Dashboard

El dashboard muestra:
- **Estadísticas de usuarios**: Total, activos, nuevos este mes
- **Estadísticas de productos**: Total, activos, stock bajo
- **Categorías activas**

### Paso 3: Gestionar Productos

1. **Crear tu primer producto**
   - Ve a la pestaña **"Productos"**
   - Haz clic en **"Nuevo Producto"**
   - Completa el formulario:
     - Nombre
     - Descripción
     - Precio
     - Categoría
     - Tallas (separadas por comas: S, M, L)
     - Stock
     - Género
     - Imagen
   - Haz clic en **"Crear Producto"**

2. **Editar un producto**
   - En la lista, haz clic en el icono de editar
   - Modifica los campos necesarios
   - Guarda los cambios

3. **Activar/Desactivar productos**
   - Usa el toggle para cambiar el estado
   - Los productos desactivados no aparecen en el catálogo público

### Paso 4: Gestionar Categorías

1. **Crear categoría**
   - Ve a la pestaña **"Categorías"**
   - Haz clic en **"Nueva Categoría"**
   - Ingresa nombre y descripción
   - Guarda

2. **Editar/Eliminar categorías**
   - Similar al proceso de productos

### Paso 5: Gestionar Usuarios

1. **Ver lista de usuarios**
   - Ve a la pestaña **"Usuarios"**
   - Revisa la lista completa

2. **Cambiar rol de usuario**
   - Selecciona un usuario
   - Haz clic en **"Cambiar Rol"**
   - Selecciona Admin o Cliente
   - Confirma

3. **Activar/Desactivar usuarios**
   - Usa el toggle para cambiar el estado

### Paso 6: Generar Reportes

1. **Reporte de ventas**
   - Ve a la pestaña **"Reportes"**
   - Selecciona **"Análisis de Ventas"**
   - Define rango de fechas (si aplica)
   - Haz clic en **"Generar Reporte"**
   - Descarga en PDF o CSV

2. **Reporte de movimientos de productos**
   - Selecciona **"Movimientos de Productos"**
   - Genera y descarga

### Paso 7: Monitorear Estado del Sistema

1. **Ver estado de servicios**
   - Ve a la pestaña **"Estado"**
   - Revisa el estado de:
     - Backend API
     - Base de Datos
     - Servicio de IA
   - Haz clic en **"Actualizar"** para verificar

### Checklist de Primeros Pasos (Administrador)

- [ ] Acceder al panel de administración
- [ ] Revisar el dashboard
- [ ] Crear un producto de prueba
- [ ] Editar el producto creado
- [ ] Crear una categoría
- [ ] Ver lista de usuarios
- [ ] Generar un reporte
- [ ] Verificar estado de servicios
- [ ] Cambiar rol de un usuario (si es necesario)

## Onboarding para Desarrolladores

### Paso 1: Configurar el Entorno de Desarrollo

1. **Requisitos previos**
   - Instalar Node.js 18+
   - Instalar Python 3.11+
   - Instalar Git
   - Crear cuenta en Supabase (para base de datos)

2. **Clonar el repositorio**
   ```bash
   git clone [url-del-repositorio]
   cd probador_virtual
   ```

3. **Configurar Frontend**
   ```bash
   cd frontend
   npm install
   cp .env.example .env
   # Editar .env con tus variables
   npm run dev
   ```

4. **Configurar Backend**
   ```bash
   cd backend
   npm install
   cp .env.example .env
   # Editar .env con tus variables
   npx prisma generate
   npx prisma migrate dev
   npm run start:dev
   ```

5. **Configurar Servicio de IA**
   ```bash
   cd python
   python -m venv env
   source env/bin/activate
   pip install -e .
   cp env.example .env
   # Editar .env con GEMINI_API_KEY
   python run_api.py
   ```

### Paso 2: Entender la Arquitectura

1. **Leer la Guía Técnica**
   - Revisa `GUIA_TECNICA.md`
   - Entiende la estructura del proyecto
   - Familiarízate con el stack tecnológico

2. **Explorar el código**
   - Revisa la estructura de carpetas
   - Lee los archivos principales
   - Entiende el flujo de datos

### Paso 3: Realizar tu Primera Modificación

1. **Crear un componente simple**
   - En `frontend/src/components/`, crea un componente de prueba
   - Úsalo en una página

2. **Agregar un endpoint**
   - En `backend/src/`, crea un nuevo módulo
   - Agrega un endpoint simple
   - Prueba con Postman o similar

3. **Ejecutar tests**
   ```bash
   # Frontend
   cd frontend && npm test

   # Backend
   cd backend && npm test

   # Python
   cd python && pytest
   ```

### Paso 4: Contribuir al Proyecto

1. **Crear una rama**
   ```bash
   git checkout -b feature/mi-nueva-funcionalidad
   ```

2. **Desarrollar**
   - Implementa tu funcionalidad
   - Escribe tests
   - Documenta tu código

3. **Commit y Push**
   ```bash
   git add .
   git commit -m "Descripción de cambios"
   git push origin feature/mi-nueva-funcionalidad
   ```

4. **Crear Pull Request**
   - Abre un PR en el repositorio
   - Describe los cambios
   - Espera revisión
